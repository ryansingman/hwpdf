# hwpdf
Create pdf from images in a directory

## Usage
```
usage: hwpdf.py [-h] [--uname UNAME] [--src_dir SRC_DIR] [--dst_dir DST_DIR] [--proj_name PROJ_NAME]

optional arguments:
  -h, --help            show this help message and exit
  --uname UNAME         username to prepend to output file name
  --src_dir SRC_DIR     directory to get images from
  --dst_dir DST_DIR     directory to place pdf and tex file in
  --proj_name PROJ_NAME
                        name of project, to use for output filenames
```
